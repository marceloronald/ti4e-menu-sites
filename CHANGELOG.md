# Changelog

Todas as mudanças relevantes deste projeto serão documentadas aqui.

O formato segue [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/)
e o versionamento segue [Semantic Versioning](https://semver.org/lang/pt-BR/).

---

## [1.0.0]

### Added
- Estrutura inicial do repositório
- Snippet HTML para versão CSS puro e Tailwind CSS
- Arquivo JS comentado com TODOs para implementação
- README com instruções de uso, variáveis CSS e classes JS
- Classe `.org-menu__link--active` no CSS puro para indicar o site atual no dropdown

### Changed
- Label do grupo substituído por link ativo (`aria-current="page"`) nas versões CSS e Tailwind
- URLs dos links atualizadas para os sites reais do Programa PotencializEE
- Logo atualizado para `logo-minimal-white.svg` (adequado ao fundo azul do botão)
- Label do botão corrigido para "Sites do PotencializEE" na versão Tailwind

### Removed
- Classe `.org-menu__group-label` e variáveis CSS associadas (substituída pelo link ativo)
